# 🚀 Mini CRM - Lead Management System

## 📌 Project Overview
This is a **Mini CRM (Customer Relationship Management)** system built using the MERN stack.  
It allows users to manage leads by adding, updating, and deleting records.

---

## ✨ Features
- ➕ Add new leads
- 📋 View all leads
- 🔄 Update lead status (New / Contacted / Converted)
- ❌ Delete leads
- 💾 Data stored in MongoDB

---

## 🛠 Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js + Express.js
- Database: MongoDB
- API Testing: Thunder Client / Postman
